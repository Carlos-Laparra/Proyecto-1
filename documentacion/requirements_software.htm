Software Requirements

Revisar si su PC cumple con los requerimientos mínimos para poder jugar ColorMania

 
Prioridad El usuario, a través de un txt, crea una serie de cuadros los cuales podrá elegir entre ordenar o hacer que el programa los ordene automáticamente. Para ambos casos se debe encontrar el algoritmo que logre dicha funcionalidad, regresando así la serie de cuadros ordenados entre columnas.
Propósito Lo que pretende el programa es medir la capacidad del programador para crear un algoritmo que ordene de manera adecuada los colores. Así mismo se mide la capacidad de usar listas a su favor, aplicando los conceptos de pila y cola para poder ordenar de manera adecuada.
Requerimientos Archivo de texto que contiene como estarán ordenados los colores
Operación Ordenar entre columnas
Resultado Juego ordenado, mostrando paso a paso al usuario como lo realizó, o regresando el txt ordenado.
